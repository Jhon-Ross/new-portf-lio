Projetos

DealHunter

DealHunter é uma plataforma desenvolvida em Python que utiliza Web Scraping para localizar automaticamente as melhores ofertas em sites de e-commerce. Com o uso do Selenium em modo headless, a ferramenta navega de forma eficiente e extrai dados em tempo real, eliminando a necessidade de buscas manuais. Ideal para quem deseja acompanhar promoções de forma prática e automatizada.
